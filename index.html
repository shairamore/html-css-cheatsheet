Basic HTML structure

There are no strict rules when it comes to HTML, but it is important to know and understand the basic structure to create a functional website. It is also important to understand the hierarchy and arrangement of different HTML tags for your page to render properly.

HTML tags

HTML tags are keywords that helps the web browser translate the code into a reader-friendly format and determines how the code is displayed to a web page. HTML tags contain three parts: opening tag, content, and closing tag. The syntax is as follows:

<opening tag>content</closing tag>

Here are the most basic HTML tags:

Tag Name Description
<!DOCTYPE html> HTML document type declaration The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in. It is the first line of code required in every HTML or XHTML document.
<html> HTML tag The root element that makes up a web page. Everything else in your web page are wrapped between the HTML tag, otherwise your content will not show.
<head> Head Metadata such as the webpage title and CSS links are stored between this tag.
<body> Body All of the displayable content such as headlines, paragraphs, and images are written between these tags otherwise it will not be displayed on your web page.
<div> Division / Section Used to section contents of your web page.
<h1> Heading 1 The highest level of heading which has the largest font size in the HTML document. This is usually used for headlines and displaying the web page title. Typically followed by <h2>, <h3>, up to <h6> tags.
<p> Paragraph Displays blocks of text.

Reminder: do not forget to put the closing tags after the opening tag. Not putting closing tags will mess up your code.
